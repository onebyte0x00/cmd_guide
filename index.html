Most Used Commands

ECHO

Display messages or enable/disable command echoing

ECHO Hello World
@ECHO OFF

FOR

Loop through files, directories, or text

FOR %i IN (*.txt) DO ECHO %i
FOR /L %i IN (1,1,10) DO ECHO %i

IF

Conditional statement execution

IF "%var%"=="value" ECHO Match
IF EXIST file.txt DEL file.txt

SET

Display or set environment variables

SET var=value
SET /A result=5+3

Getting Started with Batch Scripts

  1. Create a file with .bat or .cmd extension
  2. Start with @ECHO OFF to hide commands
  3. Add your commands (one per line)
  4. Run by double-clicking or from command prompt

Example Script

@ECHO OFF
:: This is a comment
ECHO Starting backup...
XCOPY "C:\source" "D:\backup" /E /I
IF ERRORLEVEL 1 (
    ECHO Backup failed!
) ELSE (
    ECHO Backup completed successfully
)
PAUSE
