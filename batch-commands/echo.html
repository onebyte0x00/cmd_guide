ECHO Command

Syntax

ECHO [ON | OFF]
ECHO [message]

Description

Displays messages or turns command echoing on or off.

Examples

@ECHO OFF
ECHO Hello World!
ECHO.
ECHO Current date is: %DATE%

Notes

  • Use @ECHO OFF at the beginning of scripts to hide commands
  • Add . after ECHO to display empty line: ECHO.
