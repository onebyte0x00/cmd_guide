ECHO Command

Syntax

ECHO [ON | OFF]
ECHO [message]

Description

Displays messages or turns command echoing on or off.

  • ECHO ON - Displays each command before execution (default)
  • ECHO OFF - Stops displaying commands (but not outputs)
  • @ECHO OFF - Hides the ECHO OFF command itself
  • ECHO message - Displays the specified message
  • ECHO. - Displays an empty line

Examples

Basic Usage

@ECHO OFF
ECHO Hello World!
ECHO Current date: %DATE%
ECHO.

Creating a Blank Line

ECHO Line 1
ECHO.
ECHO Line 2

Special Characters

ECHO ^& ^| ^< ^> ^^

Output: & | < > ^

Notes

  • To display special characters, prefix them with a caret (^)
  • Use @ prefix to hide individual commands
  • ECHO without parameters shows the current echo state
