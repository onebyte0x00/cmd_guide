IF Command

Syntax

IF [NOT] ERRORLEVEL number command
IF [NOT] string1==string2 command
IF [NOT] EXIST filename command
IF [NOT] DEFINED variable command
IF [NOT] CMDEXTVERSION number command
IF [/I] string1 compare-op string2 command

Description

Performs conditional processing in batch programs.

  • NOT - Reverses the condition
  • ERRORLEVEL - Checks exit code
  • == - String comparison
  • EXIST - File existence check
  • DEFINED - Variable existence check
  • /I - Case-insensitive comparison

Examples

Basic Conditions

IF "%OS%"=="Windows_NT" ECHO Running on Windows NT
IF NOT EXIST "C:\file.txt" ECHO File not found

Error Level Check

SOME_COMMAND
IF ERRORLEVEL 1 (
    ECHO Command failed with error %ERRORLEVEL%
    EXIT /B 1
)

Nested IF

IF "%1"=="" (
    ECHO Missing parameter
) ELSE IF "%1"=="/?" (
    ECHO Display help
) ELSE (
    ECHO Processing with %1
)

Case Insensitive

IF /I "%choice%"=="YES" (
    ECHO You said yes
)

  • Always quote variables: IF "%var%"=="value"
